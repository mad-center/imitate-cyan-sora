# imitate-cyan-sora

该项目为《仿青空》MAD教程归档。目的是为了给新手提供一定的思路和实践帮助。

## 软件准备

- AE CC 2020 +
- Red Gaint TrapCode 15
- Red Magic Bullet Suite 14
- Video Copilot Optical Flares v1.3.5. VR

## 教程说明

- /manuscript

  该文件夹为个人对七日课吧《仿青空》教程初步模仿的笔记。

- projects

    - imitate-cyan-sora

      为个人对七日课吧《仿青空》教程初步模仿的AE工程文件参考。

      > 请注意：该项目非常粗糙，只面向初学者，只关注功能实现，不关注工程文件组织和规范。

    - AE-仿青空-纵使日薄西山

      为个人第二次仿青空的工程文件参考，该文件参考了文献[2]的工程文件，并根据自身的所用素材对工程文件做出了适当的调整，包含分镜重写，调色，删减等。同时，该文件的目录组织具有一定的规范，可以作为参考。
